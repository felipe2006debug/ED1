# 📚 Estudos de Estrutura de Dados & Algoritmos

Este repositório documenta minha jornada de estudos e implementação de estruturas de dados fundamentais, utilizando C como base para o entendimento de gerenciamento de memória e lógica de baixo nível.

---

## 🛠️ O que já exploramos

Ao longo desta trajetória, focamos em compreender não apenas a implementação, mas a complexidade e o uso prático de cada estrutura:

### 1. Estruturas Lineares
* **Listas Encadeadas (Linked Lists):**
    * Implementação de listas simples e duplamente encadeadas.
    * Gestão dinâmica de memória e manipulação de ponteiros.
* **Pilhas (Stacks):**
    * Conceito LIFO (Last-In, First-Out).
    * Aplicação em expressões matemáticas e controle de chamadas de funções.
* **Filas (Queues):**
    * Conceito FIFO (First-In, First-Out).
    * Implementação básica e filas circulares.

### 2. Algoritmos & Lógica
* **Manipulação de Memória:** Foco em `malloc`, `free` e prevenção de vazamentos (*memory leaks*).
* **Lógica de Circuitos:** Exercícios de Booleanos e diagramas lógicos aplicados à computação.
* **Análise de Desempenho:** Discussão sobre complexidade de tempo e espaço (Big O notation).
